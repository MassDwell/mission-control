## Technical Decisions

- **Pivoted from Vite+React to static site:** NPM cache ownership issues blocked npm install. Went pure HTML/JS with CDN dependencies instead.
- **Alpine.js for reactivity:** Lightweight, no build step, perfect for dashboard interactivity
- **Chart.js for visualizations:** Pipeline funnels, revenue charts, trend lines
- **Tailwind CSS via CDN:** Rapid styling without build complexity
